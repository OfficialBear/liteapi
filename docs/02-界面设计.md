# 界面设计

## 布局

```
┌──────────┬──────────────────────────────────────────┐
│          │ [Tab1] [Tab2] [Tab3] [+]                 │
│ 侧边栏   ├──────────────────────────────────────────┤
│          │ [Method ▼] [URL _________________] [Send]│
│ 搜索     │ [Save] [</>] [···]                       │
│          ├──────────────────────────────────────────┤
│ 📁 用户   │ [Params] [Headers] [Cookies] [Auth] [Body]│
│  ├ GET .. │                                          │
│  └ POST . ├──────────────────────────────────────────┤
│ 📁 订单   │ 200 OK  234ms  1.2KB  [Body][Headers]... │
│          │ [Pretty] [Raw] [Preview]          [📋]   │
│          │ { "id": 1, "name": "..." }               │
│ [EN] [🌙] │                                          │
└──────────┴──────────────────────────────────────────┘
```

## 五大区域

| 区域 | 组件 | 说明 |
|------|------|------|
| 侧边栏 | Sidebar → SidebarToolbar + FolderTree | 文件夹树、搜索、语言/主题切换 |
| 标签栏 | TabBar | 标签页管理，浏览器风格 "+" 按钮 |
| 请求编辑 | RequestEditor → RequestBasicInfo + RequestSubTabs | URL 栏 + 参数标签页 |
| 响应查看 | ResponseViewer → ResponseBodyPanel | 状态栏 + 标签 + 响应体 |
| 侧边栏/内容分隔 | ResizeHandle | 可拖拽调整宽度 |

## 交互原则

- **一行搞定**：Save、Send、CodeGen、Details 全在 URL 栏一行
- **渐进展开**：名称/描述默认隐藏，点 `···` 展开
- **固定头部**：响应状态栏和标签栏固定不滚动
- **直接操作**：文件夹拖拽排序，侧边栏拖拽调整宽度
- **减少选择**：方法从 7 个减为 5 个（去掉 HEAD/OPTIONS），Body 从 7 种减为 5 种

## 组件树

```
App
├── Sidebar
│   ├── SidebarToolbar (搜索 + 新建按钮)
│   ├── FolderTree (Ant Tree + 右键菜单 + 拖拽)
│   └── Footer: 语言 + 主题（侧边栏底部）
├── SidebarDivider (可拖拽)
├── TabBar (Ant Tabs + 右键菜单)
├── RequestEditor
│   ├── RequestBasicInfo (Method + URL + Send + Save + CodeGen + Details)
│   └── RequestSubTabs
│       ├── ParamsPanel (Path只读 + Query可编辑)
│       ├── HeadersPanel (KeyValueTable)
│       ├── CookiesPanel (Name/Value Table)
│       ├── AuthPanel (NoAuth/Bearer/Basic)
│       └── BodyPanel (JSON/FormData/UrlEncoded/Raw/Binary)
├── ResponseViewer
│   ├── Header (状态码 + 耗时 + 大小 + 标签)
│   └── ResponseBodyPanel
│       ├── Segmented (Pretty/Raw/Preview + 复制)
│       └── Body (可滚动)
```

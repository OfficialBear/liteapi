# 接口设计 — Tauri Commands

## 架构

前端（React + Zustand）通过 `invoke()` 调用 Rust 后端命令。Tauri 不可用时回退到 mock 数据。

```
┌──────────────┐    invoke()     ┌──────────────────┐
│   React UI   │ ───────────────→│  Rust Backend    │
│  (tauri-api) │ ←───────────────│  (commands.rs)   │
└──────────────┘    Result<T>    └───────┬──────────┘
                                         │
                                    ┌────▼─────┐
                                    │  SQLite   │
                                    │ (rusqlite)│
                                    └──────────┘
```

## 命令列表

### 数据库初始化
```
init_app() → InitData { folders, apis, tabs, settings }
```

### 文件夹 CRUD
```
create_folder(id, parentId?, name) → Folder
update_folder(id, name?, parentId?, sort?) → void
delete_folder(id) → void
copy_folder(sourceId, newParentId?, newName) → Folder
```

### API CRUD
```
create_api(id, folderId?, name, method, url) → Api
update_api(id, name?, method?, url?, folderId?) → void
delete_api(id) → void
copy_api(sourceId, targetFolderId?, newName) → Api
```

### 标签页持久化
```
save_tabs(tabs: TabRow[]) → void
```

### 设置
```
get_setting(key) → string | null
set_setting(key, value) → void
```

### 数据导入导出
```
export_all_data() → string (JSON)
import_all_data(jsonStr) → void
```

### HTTP 请求发送
```
send_http_request(config: HttpRequestConfig) → HttpResponseData
```

**HttpRequestConfig:**
| 字段 | 类型 | 说明 |
|------|------|------|
| method | string | GET/POST/PUT/DELETE/PATCH |
| url | string | 请求 URL |
| headers | HttpKeyValue[] | 请求头 |
| query_params | HttpKeyValue[] | 查询参数 |
| body_type | string | none/json/form-data/x-www-form-urlencoded/raw/binary |
| body_content | string | 请求体内容 |
| auth_type | string | none/bearer/basic |
| auth_token | string? | Bearer token |
| auth_username | string? | Basic 用户名 |
| auth_password | string? | Basic 密码 |

**HttpResponseData:**
| 字段 | 类型 | 说明 |
|------|------|------|
| status | u16 | HTTP 状态码 |
| status_text | string | 状态文本 |
| duration | u64 | 耗时（ms） |
| size | u64 | 响应体大小（bytes） |
| headers | Map<string, string> | 响应头 |
| body | string | 响应体 |

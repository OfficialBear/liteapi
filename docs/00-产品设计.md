# 产品设计

## 产品定位

LiteAPI 是一款本地优先的桌面 API 调试工具，面向个人开发者，替代 Postman/Apipost 的核心工作流。

## 核心原则

| 原则 | 说明 |
|------|------|
| 本地优先 | 数据存储在本地 SQLite，无需注册账号，无需联网 |
| 单用户 | 为个人开发者设计，不涉及团队协作 |
| 轻量 | 安装包小，启动快，内存占用低 |
| 快速 | 原生性能（Rust + Tauri），操作响应即时 |
| 简单 | 简洁 UI，减少认知负担，专注核心流程 |

## 目标用户

- 后端开发者，日常需要调试 HTTP API
- 前端开发者，需要测试后端接口
- 个人项目，不需要团队协作功能

## 核心工作流

1. 输入 URL → 配置参数 → 发送请求 → 查看响应
2. 保存请求到集合（文件夹）以便复用
3. 管理多个请求标签页

## 不做的事

- 不做团队协作
- 不做云端同步
- 不做 API 文档生成
- 不做自动化测试
- 不做 Mock 服务
- 不做 WebSocket/GraphQL（首版）

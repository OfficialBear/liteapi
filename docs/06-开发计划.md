# 开发计划

## 已完成

按优先级排列：

| 优先级 | 事项 | 说明 |
|--------|------|------|
| **P0** | 项目骨架（Tauri + React + Vite） | 桌面壳、窗口管理、前端工程化 |
| **P0** | 请求编辑 + 发送 + 响应展示 | 核心闭环：Method/URL/Params/Headers/Body → Send → Response |
| **P0** | 数据持久化（SQLite） | 文件夹/API/标签页/响应全部持久化，关掉重开数据不丢 |
| **P0** | 文件夹树 + API CRUD | 无限层级文件夹，右键菜单，拖拽移动，搜索过滤 |
| **P1** | 多标签页管理 | 浏览器风格标签栏，关闭/复制/脏标记 |
| **P1** | HTTP 请求走 Rust 后端 | reqwest 发送，无 CORS 限制 |
| **P1** | 亮暗主题 + 中英文 | 主题跟随系统，语言切换，响应区域主题适配 |
| **P2** | 代码生成 | cURL / Fetch / Axios / Python |
| **P2** | 拖拽调整布局 | 侧边栏宽度、请求/响应比例均可拖拽 |
| **P2** | 响应体复制 | Pretty/Raw/Preview 模式下均可一键复制 |

# 数据模型

> SQLite via rusqlite (Tauri v2)。UUID 为 TEXT，时间戳 ISO 8601，布尔值 INTEGER 0/1。

## 核心表

### folders — 文件夹

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT PK | UUID |
| parent_id | TEXT FK → folders(id) | NULL = 根级 |
| name | TEXT NOT NULL | 文件夹名 |
| sort | INTEGER DEFAULT 0 | 排序 |
| created_at | TEXT | ISO 8601 |
| updated_at | TEXT | ISO 8601 |

### apis — API 接口

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT PK | UUID |
| folder_id | TEXT FK → folders(id) ON DELETE CASCADE | NULL = 根级 |
| name | TEXT NOT NULL | 接口名 |
| description | TEXT DEFAULT '' | 描述 |
| method | TEXT CHECK(method IN ('GET','POST','PUT','DELETE','PATCH')) | HTTP 方法 |
| url | TEXT DEFAULT '' | 请求 URL |
| sort | INTEGER DEFAULT 0 | 排序 |
| created_at | TEXT | ISO 8601 |
| updated_at | TEXT | ISO 8601 |

### api_params — 请求参数（Query + Headers）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT PK | UUID |
| api_id | TEXT FK → apis(id) CASCADE | |
| type | TEXT CHECK(type IN ('query','header')) | 参数类型 |
| key | TEXT NOT NULL | 参数名 |
| value | TEXT DEFAULT '' | 参数值 |
| description | TEXT DEFAULT '' | 说明 |
| enabled | INTEGER DEFAULT 1 | 0=禁用 |
| sort | INTEGER DEFAULT 0 | 排序 |

### api_cookies — Cookie

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT PK | UUID |
| api_id | TEXT FK → apis(id) CASCADE | |
| name | TEXT NOT NULL | Cookie 名 |
| value | TEXT DEFAULT '' | Cookie 值 |

### api_configs — API 级认证配置

| 字段 | 类型 | 说明 |
|------|------|------|
| api_id | TEXT PK FK → apis(id) CASCADE | |
| auth_type | TEXT CHECK(auth_type IN ('none','bearer','basic')) DEFAULT 'none' | |
| auth_token | TEXT | Bearer Token |
| auth_username | TEXT | Basic Auth 用户名 |
| auth_password | TEXT | Basic Auth 密码 |

### api_bodies — 请求体配置

| 字段 | 类型 | 说明 |
|------|------|------|
| api_id | TEXT PK FK → apis(id) CASCADE | |
| type | TEXT CHECK(type IN ('none','json','form-data','x-www-form-urlencoded','raw','binary')) | |
| content | TEXT DEFAULT '' | 原始内容 |
| binary_path | TEXT | 二进制文件路径 |

### tabs — 标签页持久化

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT PK | UUID |
| api_id | TEXT FK → apis(id) ON DELETE SET NULL | NULL=未保存 |
| sort | INTEGER DEFAULT 0 | 排序 |
| is_dirty | INTEGER DEFAULT 0 | 是否有未保存修改 |

### responses — 响应缓存（每个 API 保留最新一条）

| 字段 | 类型 | 说明 |
|------|------|------|
| api_id | TEXT PK FK → apis(id) CASCADE | |
| status_code | INTEGER | HTTP 状态码 |
| status_text | TEXT | 状态文本 |
| duration | INTEGER | 耗时（ms） |
| size | INTEGER | 响应体大小（bytes） |
| headers | TEXT | JSON 编码的响应头 |
| cookies | TEXT | JSON 编码的 Set-Cookie |
| body | TEXT | 响应体 |
| updated_at | TEXT | ISO 8601 |

### settings — 键值设置

| 字段 | 类型 | 说明 |
|------|------|------|
| key | TEXT PK | 设置键 |
| value | TEXT | JSON 或纯文本值 |

## 辅助表

- **folder_configs** — 文件夹级认证（待实现继承链）
- **folder_config_params** — 文件夹级参数
- **api_body_params** — FormData/UrlEncoded 的键值对

## 级联规则

| 操作 | 行为 |
|------|------|
| 删除文件夹 | 递归删除子文件夹 + 其下 API |
| 删除 API | 级联删除其 configs/params/cookies/bodies/responses |
| API 的 folder_id | 可为 NULL，表示根级 API，直接显示在树根 |

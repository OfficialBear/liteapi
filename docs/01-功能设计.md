# 功能设计

## 已实现

### 请求编辑
- HTTP 方法：GET、POST、PUT、DELETE、PATCH
- URL 输入，自动补全 `https://`
- 请求参数：Query（可编辑）+ Path（自动解析 URL 中 `:param`）
- 请求头（Headers），支持批量编辑
- Cookie（独立标签页，Name/Value 表格）
- 认证：No Auth / Bearer Token / Basic Auth
- 请求体：JSON、Form Data、UrlEncoded、Raw、Binary
- JSON 编辑器支持格式化、语法校验

### 发送请求
- 桌面模式：Rust 后端（reqwest），无 CORS 限制
- 浏览器开发模式：axios 回退
- 显示状态码、耗时、响应大小
- 响应头、Cookie、Body 分标签展示
- Body 支持 Pretty（JSON 美化）、Raw、Preview（文本/JSON）三种模式
- 响应体一键复制

### 集合管理
- 文件夹树，无限层级嵌套
- 右键菜单：新建子文件夹、新建 API、重命名、复制、移动、删除
- 拖拽排序和移动
- 搜索过滤

### 标签页
- 多标签页管理（浏览器风格）
- 新建标签页按钮紧贴最后一个标签
- 右键菜单：关闭、关闭其他、复制
- 修改未保存提示
- 标签标题随 API 名称同步更新

### 其他
- 中/英文切换
- 亮色/暗色主题
- 侧边栏可拖拽调整宽度
- 请求/响应区域可拖拽调整比例
- 代码生成：cURL、Fetch、Axios、Python
- [x] 快捷键（Ctrl+Enter 发送）

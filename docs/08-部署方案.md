# 部署方案

## 开发

```bash
# 仅前端（浏览器模式，HTTP 请求走 axios）
npm run dev

# 桌面应用（Tauri，HTTP 请求走 Rust reqwest）
npm run tauri
```

## 构建

```bash
# 前端构建
npm run build

# Tauri 桌面打包
npx tauri build
```

## 产物

| 平台 | 格式 |
|------|------|
| macOS | `.dmg` |
| Windows | `.msi` / `.exe` |
| Linux | `.AppImage` / `.deb` |

## 环境要求

### 开发
- Node.js 18+
- Rust 1.77+
- macOS / Windows / Linux

### 运行
- macOS 11+ / Windows 10+ / Linux（主流发行版）
- 无需额外依赖（SQLite bundled 编译）

## 目录结构（运行时）

```
~/.local/share/com.liteapi.app/   (Linux)
~/Library/Application Support/com.liteapi.app/   (macOS)
%APPDATA%/com.liteapi.app/   (Windows)
└── liteapi.db    # SQLite 数据库
```
